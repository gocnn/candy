# Contributing to candy

Thank you for your interest in contributing to `candy`!  
We welcome issues, pull requests, and ideas to make this project better.

## How to Get Started

- **Check existing issues** before opening a new one.
- **Open a new issue** to:
  - Report a bug
  - Propose a feature
  - Ask a design / API question

Please include clear steps to reproduce and as much context as possible.

## Development Setup

- Go version: see `go.mod`
- Clone the repository and create a feature branch:
  - `dev` is the main development branch.
- Run tests before submitting changes:

  ```bash
  go test ./...
  ```
